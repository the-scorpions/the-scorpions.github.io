What Is C/C++ ?

C is a very old low level Language. Despite Being old, C is one of the most widely used Languages till date due to its Lightning Fast Execution Speed. Unlike Python, C can prove to be a little Tough For Beginners To grasp and understand. The Successor of C is C++, C++ is a language built upon C with Extended OOP functionality and a verbose Standard Library to Reduce the Amount of code required.

Where Is C/C++ Used?

  1. 1. Microcontroller Programming (Arduino, Node MCU etc.)
  2. 2. Programming Drivers (Drivers for devices like Printers, Hard Drives, Graphics Cards etc.)
  3. 3. Operating Systems Programming
  4. 4. Desktop Development (Developing applications for OS like Windows, Linux etc.)
  5. 5. Game Development

Who Should Learn C/C++?

Unlike Python, C/C++ is not a language everyone must know. Over Time it has become some what of a Niche' programming language. However in tasks that require Very Fast Execution, C++ is the only reliable choice. Moreover, For Game Developers, C++ is still a highly favoured Language. Other than that, those serious about Competetive Programming, must consider C++ as their language of choice due to its low memory usage and Fast Execution.

Roadmap to Getting good at C/C++

DISCLAIMER : THIS IS BASED ON OUR PERSONAL EXPERIENCE OF OVER A YEAR AND A HALF WITH C/C++.

Like anyother language, There is no set and defined path to learning C/C++, Learn on the Go, Building projects and cool console based applications using C/C++ is possibly the best way to get good at it.

  • 1. Learn The Basic Syntax i.e Conditions, Loops, Arrays ,Functions
  • 2. Begin with Pointers and try to fully grasp the concept of Pointers and Dynamic Memory Allocation
  • 3. Learn to better structure your code using Structs and Unions
  • 4. Get started with OOP using C++
  • 5. Learn to Use the Standard Template Library
  • 6. Get Started with Basics of Data Structures
  • 7. Build Cool Shit!
