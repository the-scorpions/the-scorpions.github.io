What Is C/C++ ?

C is a very old low level Language. Despite Being old, C is one of the most widely used Languages till date due to its Lightning Fast Execution Speed. Unlike Python, C can prove to be a little Tough For Beginners To grasp and understand. The Successor of C is C++, C++ is a language built upon C with Extended OOP functionality and a verbose Standard Library to Reduce the Amount of code required.

Where Is C/C++ Used?

  1. Microcontroller Programming
  2. Programming Drivers
  3. Operating Systems Programming
  4. Desktop Development
  5. Game Development

Who Should Learn C/C++?

Unlike Python, C/C++ is not a language everyone must know. Over Time it has become some what of a Niche' programming language. However in tasks that require Very Fast Execution, C++ is the only reliable choice. Moreover, For Game Developers, C++ is still a highly favoured Language. Other than that, those serious about Competetive Programming, must consider C++ as their language of choice due to its low memory usage and Fast Execution.
