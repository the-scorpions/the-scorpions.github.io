What Are CodeJams?

Code Jams are competetive programming events. These test the ability of a coder to come up with solution of a problem using their knowledege of code. Unlike hackathons, the Code Jam problems are more theoretical and require the Participants to have a clear knowledge of the concepts behind the problem.

Which Language to Choose ?

It is our humble advice to stick with the Language you know the best. Most codejams have no restrictions on Language. It is possible to solve any problem with any Language, what matters is how good are you with that particular Language. Some Common Languages prefer to use are :

  • C/C++
  • Python
  • Java
  • JavaScript
  • C#
